# 경쟁사 목록 및 포지셔닝

## 경쟁 구도

```
            도구/계산기 특화
                 ↑
    Smogon   |  Poke-Korea(목표)
    Showdown |  yakkun
             |
정보 위키 ←──┼──→ 대전 메타
             |
    PokemonDB|  Pikalytics
    Serebii  |
                 ↓
            정보/데이터 특화
```

## 직접 경쟁사

| 사이트           | URL                 | 주력             | 강점                     | 약점          | 위협 수준 |
| ---------------- | ------------------- | ---------------- | ------------------------ | ------------- | --------- |
| yakkun           | yakkun.com          | 내구/스피드 조정 | 일본 대전 유저 표준 도구 | 한국어 미지원 | 높음      |

| Pokemon Showdown | pokemonshowdown.com | 배틀 시뮬레이터  | 데미지 계산기 표준       | 영어 중심     | 중간      |

## 한국어 시장 경쟁

| 사이트      | 유형 | 강점                     | 약점                   |
| ----------- | ---- | ------------------------ | ---------------------- |
| 나무위키    | 위키 | 검색 장악력, 방대한 정보 | 도구 없음, 비공식      |
| 포켓몬 위키 | 위키 | 공식 데이터              | 도구 없음, 트래픽 낮음 |

## poke-korea의 차별화 포인트

1. **한국어 전용 도구**: 한국어로 포켓몬 도구를 제공하는 유일한 사이트
2. **한국 메타 반영**: 한국 대전 환경에 맞춘 분석
3. **모바일 최적화**: 게임 중 모바일 사용에 최적화된 UI
4. **통합 도구**: 데미지 계산 + 도감 + 능력치 계산을 한 곳에서

## 기회 영역

- 한국어 포켓몬 도구 검색 시 경쟁이 거의 없음
- 나무위키는 도구 제공 불가 → 도구 관련 검색은 블루오션
- 일본(yakkun) 수준의 도구를 한국어로 제공하면 대전 유저 흡수 가능
