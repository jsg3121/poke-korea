# 타겟 사용자 정의

## 1차 타겟: 한국 포켓몬 대전 유저

| 항목      | 설명                                               |
| --------- | -------------------------------------------------- |
| 연령대    | 10대 후반 ~ 30대                                   |
| 특성      | 포켓몬 대전(랭크배틀, VGC)에 적극 참여             |
| 니즈      | 빠른 데미지 계산, 포켓몬 정보 조회, 메타 분석      |
| 사용 패턴 | 대전 준비 시 집중 사용, 시즌 초/중반에 트래픽 증가 |
| 사용 기기 | 모바일(게임 중) + 데스크톱(팀 빌딩 시)             |

## 2차 타겟: 한국 포켓몬 일반 유저

| 항목      | 설명                                   |
| --------- | -------------------------------------- |
| 연령대    | 10대 ~ 40대                            |
| 특성      | 포켓몬 스토리/도감 수집 중심 플레이    |
| 니즈      | 포켓몬 도감 정보, 진화 조건, 기술 목록 |
| 사용 패턴 | 새 게임 출시 전후 트래픽 급증          |
| 사용 기기 | 모바일 중심                            |

## 3차 타겟: 포켓몬 관심 유저 (잠재 유저)

| 항목      | 설명                                             |
| --------- | ------------------------------------------------ |
| 연령대    | 전 연령                                          |
| 특성      | 검색을 통해 유입, 포켓몬에 대한 일반적 궁금증    |
| 니즈      | "포켓몬 타입 상성", "최강 포켓몬" 등 검색형 정보 |
| 사용 패턴 | 단발성 방문, 이탈률 높음                         |
| 사용 기기 | 모바일 중심                                      |

## 세그먼트별 전략 방향

| 세그먼트        | 핵심 전략                | 주요 지표                |
| --------------- | ------------------------ | ------------------------ |
| 1차 (대전 유저) | 도구 고도화, 리텐션 강화 | D7 리텐션, 도구 사용률   |
| 2차 (일반 유저) | 콘텐츠 확대, SEO 강화    | 검색 유입, 페이지뷰/세션 |
| 3차 (잠재 유저) | SEO + 온보딩 개선        | 이탈률 감소, 활성화율    |
