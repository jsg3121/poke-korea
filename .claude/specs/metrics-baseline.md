## 핵심 관찰 (분석 기준점)

1. **검색 유입 비중 83%** — SEO가 사실상 유일한 유입 경로. 어느 한 검색엔진 의존도가 높음 (구글 40.5%, 네이버 36%)
2. **/type-effectiveness가 사이트 트래픽의 절반 이상을 차지** — 단일 페이지 의존도 위험
3. **DAU/MAU 4.2%** — 재방문 기반 약함. 검색 → 정보 확인 → 이탈 패턴
4. **CTR 2.29%, 평균 순위 6.8** — 6위에서 정상적인 CTR(7~10%)에 한참 못 미침. 메타 최적화 여지 큼
5. **모바일 55.6% + 네이버 모바일 30.1%** — 모바일 + 네이버 SEO가 핵심 레버
6. **챔피언스 페이지군 (/champions/*)** — 30일 사용자 합계 3,494명. 최근 SEO 개선했음에도 노출 저조
7. **네이버 데이터: 특수 폼 키워드(거다이맥스/메가/이로치)에서 노출 다수, CTR 저조** — 메타 최적화의 즉시 개선 효과가 가장 큰 영역
