> 챔피언스 페이지군의 본질 파악 (종착점 vs 인입 경로) — 광고 재도입 의사결정 근거

| 시작점          | 총 진입 | 다음 페이지 이동                 | 이탈(세션 종료) | 외부 이동(`/list`, `/detail`, `/moves` 등) |
| --------------- | ------- | -------------------------------- | --------------- | ------------------------------------------ |
| /champions/list | 317     | 133 (42%, 챔피언스 내부)         | **184 (58%)**   | ~10 (3%)                                   |
| /champions/tier | 565     | 91 (16%, 챔피언스 내부) + 37 → / | **437 (77%)**   | ~50 (9%)                                   |
| /champions      | 772     | 290 (38%, 챔피언스 내부)         | **482 (62%)**   | ~20 (3%)                                   |

**핵심 발견**:

1. **챔피언스 페이지는 종착점 성격이 매우 강함** (이탈률 58~77%)
2. **외부 광고 페이지로 인도하는 비율은 3~9%로 낮음** → 광고 도입 시 다른 페이지 수익 손실 우려 미미
3. **챔피언스 내부 순환은 16~42%** → 챔피언스 내에서 광고가 반복 노출되어도 누적 노출 효과 기대 가능

### 1.5 핵심 인사이트 (갱신)

#### 검증된 사실

1. **인아티클 형식이 CTR 4~6배 우수**: 기술 상세 하단 인아티클 1.84%, PC 상성 계산기 상단 인아티클 0.49% vs 디스플레이 평균 0.05~0.3%
2. **상단 배너는 전반적으로 무력화**: 모바일 상세 상단 0.06%, 기술 상세 상단 모바일 0%, 기술도감 상단 PC 0%
3. **하단 배너는 효과 있음**: 모바일 상세 페이지 광고 하단 배너 CTR 1.30%로 가장 안정적
4. **사이드 배너 실패**: 데스크톱 상세 사이드 배너 CTR 0.03% → 구조적 문제

#### 4월 → 5월 변화 분석

- **수익 자체 성장**: 4/18 약 $9.8 (50슬롯) → 5/18 약 $11.0 (49슬롯) (+12%)
- **다만 4월 작업과의 직접 인과관계는 약함**:
  - 5월 개선된 슬롯(모바일 상세 하단, PC 인아티클, 기술 상세 인아티클)은 모두 4월에 변경 없음
  - 자연 트래픽 증가 + AdSense 자체 최적화 + 트래픽 증대 계획서 1.39.0~1.40.0 효과(메타 description, TypeChip 링크화, 챔피언스 인입 경로 추가) 영향이 더 큼
- **4월 작업의 부정적 신호**: 신규 리스트 인피드 광고가 저조 (PC $0.07/CTR 0.09%, 모바일 $0.20/CTR 0.03%, Coverage 22%/41%)

#### 기회

1. **인아티클 형식 확대**: 검증된 CTR 패턴 → 다른 페이지에도 적용
2. **챔피언스 페이지군 광고 재도입**: 트래픽 5,068 조회/월 (메인 3개) + 1,800 조회/월 (상세), 이탈률 60~77%로 종착점 성격 강함 → 광고 추가 시 순수익 증가 명확
3. **모바일 상세 상단/기술 배너 인아티클 전환**: 디스플레이 CTR 0.06%/0% → 인아티클 1.8% 패턴 적용 가능
4. **리스트 페이지 인피드 광고 제거 검토**: 4월 추가했으나 효과 없음, 사용자 스크롤 경험만 저하
