### 1.3 핵심 인사이트

#### 문제점

1. **상단 배너 CTR 저조**: 모바일/PC 상세 페이지 상단 배너 CTR 0.05~0.07% (업계 평균 0.3~0.5% 대비 매우 낮음)
2. **Coverage 낮음**: 일부 슬롯 Coverage 29~40% (광고 채워지지 않음)
3. **클릭 0 슬롯**: 모바일 상세 기술 배너, 모바일 상성계산기 하단 등 클릭 전무
4. **퀴즈 광고 비효율**: 노출 적고 수익 미미 ($0.00~$0.02)

#### 기회

1. **상성 계산기**: RPM 0.37로 최고, 트래픽 4위 → 확대 가치
2. **기술 상세 하단 인아티클**: CTR 1.49%로 최고 → 다른 페이지에도 적용
3. **리스트 페이지**: 트래픽 1위, 체류시간 211초 → 인피드 광고 확대
4. **홈페이지**: 수익 $2,093 최고 → 현재 배치 효과적

## 2. 개선 작업 계획

### Phase 1: 즉시 개선 (1-2일)

#### 1.1 CTR 0% 슬롯 제거 또는 교체

- [ ] `모바일 상세 기술 배너` - 위치/크기 변경 또는 제거
- [ ] `모바일 상성계산기 하단 배너` - 인아티클로 교체
- [ ] `PC 상성계산기 하단 배너` - 위치 조정

#### 1.2 고성과 광고 형식 확대

- [ ] **인아티클 광고** 추가 (CTR 1.49% 검증됨)
  - 상세 페이지 섹션 사이
  - 리스트 페이지 스크롤 중간
- [ ] **인피드 광고** 확대 (RPM 0.17로 양호)
  - 리스트 페이지 20개마다 1개
  - 기술/특성 목록 페이지

### Phase 2: 레이아웃 최적화 (3-5일)

#### 2.1 상세 페이지 광고 재배치

현재 스크린샷 분석 결과:

- 상단 배너가 콘텐츠와 너무 붙어 있음 → 여백 추가
- 사이드 배너 가시성 낮음 → Sticky 적용

작업:

- [ ] 상단 배너 상하 여백 24px → 40px
- [ ] 사이드 배너 `position: sticky` 적용
- [ ] 섹션 간 인아티클 광고 삽입 (기본정보↔Z기술, 기술↔타입상성)

#### 2.2 리스트 페이지 인피드 광고

- [ ] 무한스크롤 중간에 네이티브 광고 삽입
- [ ] 광고 카드를 포켓몬 카드와 유사한 스타일로 통일

#### 2.3 모바일 광고 크기 최적화

- [ ] 320x100 → 336x280 (더 높은 RPM)
- [ ] Anchor 광고 검토 (페이지 하단 고정)

### Phase 3: 고급 최적화 (1-2주)

#### 3.1 Lazy Loading 광고

- [ ] 뷰포트 진입 시 광고 로드
- [ ] Coverage 개선 및 페이지 속도 향상

#### 3.2 A/B 테스트 설정

- [ ] 광고 위치별 수익 비교
- [ ] 광고 크기별 CTR 비교

#### 3.3 퀴즈 페이지 광고 전략 재검토

- [ ] 퀴즈 진행 중 광고 제거 (UX 우선)
- [ ] 결과 페이지에만 집중 배치
- [ ] 보상형 광고 검토 (힌트 보기 시 광고 시청)

## 4. 작업 우선순위

```
1. [높음] CTR 0% 슬롯 제거/교체 → 낭비 제거
2. [높음] 리스트 페이지 인피드 광고 → 트래픽 1위 활용
3. [중간] 상세 페이지 인아티클 추가 → 검증된 형식 확대
4. [중간] 사이드 배너 Sticky 적용 → 가시성 향상
5. [낮음] 퀴즈 광고 재구성 → 트래픽 적음
6. [낮음] A/B 테스트 → 기반 작업 완료 후
```
