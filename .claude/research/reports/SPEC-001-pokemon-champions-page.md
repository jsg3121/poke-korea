## 2. 페이지별 상세

### 2.1 `/champions` — 메인 허브

| 항목   | 내용                                                            |
| ------ | --------------------------------------------------------------- |
| 목적   | 포챔스 콘텐츠 진입점, SEO 랜딩                                  |
| 콘텐츠 | 게임 소개, VP 시스템 설명, 인기 포켓몬 TOP 10, 하위 페이지 링크 |
| API    | 내부 API (인기 포켓몬 조회)                                     |

### 2.2 `/champions/list` — 전용 도감

| 항목   | 내용                                                                       |
| ------ | -------------------------------------------------------------------------- |
| 목적   | 챔피언스 수록 186종만 필터링하여 표시                                      |
| 콘텐츠 | 포켓몬 목록, 폼 통합 표시 (메가/거다이맥스/테라스탈), VP별 스탯 시뮬레이션 |
| API    | 내부 API (기존 도감 + 챔피언스 필터)                                       |

**필요 데이터:**

- 챔피언스 수록 186종 ID 목록 (신규)
- 기존 포켓몬 스탯/타입/특성 (기존 DB)

### 2.3 `/champions/calculator` — VP 데미지 계산기

| 항목   | 내용                                                       |
| ------ | ---------------------------------------------------------- |
| 목적   | VP 배분에 따른 데미지 시뮬레이션                           |
| 콘텐츠 | 공격/방어 포켓몬 선택, VP 슬라이더, 기술 선택, 데미지 결과 |
| API    | 내부 API (스탯, 기술 위력, 타입 상성)                      |

**필요 데이터:**

- VP → 스탯 변환 공식 (신규)
- 기술별 위력/타입/분류 (기존 DB)
- 타입 상성 배율 (기존)

### 2.4 `/champions/tier` — 티어 리스트

| 항목   | 내용                                        |
| ------ | ------------------------------------------- |
| 목적   | 현재 메타 포켓몬 순위 제공                  |
| 콘텐츠 | S/A/B/C/D 등급, 사용률/승률, 추천 빌드 링크 |
| API    | 외부 API (Pikalytics) 또는 수동 관리        |

**필요 데이터:**

- 사용률/승률 통계 (외부 - Pikalytics)
- 티어 등급 (수동 또는 자동 계산)

### 2.5 `/champions/teams` — 추천 팀 구성

| 항목   | 내용                                                |
| ------ | --------------------------------------------------- |
| 목적   | 아케타입별 샘플 파티 제공                           |
| 콘텐츠 | 날씨팀/트릭룸/하이퍼어택 등 6마리 조합, 시너지 설명 |
| API    | 내부 API (팀 데이터) + 외부 참조                    |

**필요 데이터:**

- 추천 팀 구성 (신규 - 수동 입력 또는 커뮤니티 수집)

## 3. API 요구사항 정리

### 3.1 내부 API (poke-korea-server) — 기존 활용

| API                | 용도                           | 상태       |
| ------------------ | ------------------------------ | ---------- |
| `getPokemonList`   | 포켓몬 목록 조회               | ✅ 기존    |
| `getPokemonDetail` | 포켓몬 상세 (스탯, 특성, 기술) | ✅ 기존    |
| `pokemon_stats`    | 능력치 데이터                  | ✅ 기존 DB |
| `pokemon_skills`   | 기술 데이터                    | ✅ 기존 DB |
| `pokemon_ability`  | 특성 데이터                    | ✅ 기존 DB |

### 3.2 내부 API (poke-korea-server) — 신규 필요

| API                         | 용도                              | 우선순위 |
| --------------------------- | --------------------------------- | -------- |
| `getChampionsPokemonList`   | 챔피언스 수록 186종 필터링        | 🔴 높음  |
| `getChampionsPokemonDetail` | 챔피언스 전용 상세 (VP 스탯 포함) | 🔴 높음  |
| `calculateVPDamage`         | VP 기반 데미지 계산               | 🟡 중간  |
| `getChampionsTier`          | 티어 리스트 조회                  | 🟢 낮음  |
| `getChampionsTeams`         | 추천 팀 조회                      | 🟢 낮음  |

### 3.3 신규 DB 테이블 — 재검토 결과

| 테이블              | 필요 여부 | 이유                                         |
| ------------------- | --------- | -------------------------------------------- |
| `champions_pokemon` | ✅ 필요   | 186종 ID 목록은 외부 소스 없음, 직접 관리    |
| `champions_tier`    | ❌ 불필요 | 외부 JSON 소비 가능 (eurekaffeine scraper)   |
| `champions_teams`   | ❌ 불필요 | Champions Lab 오픈소스 데이터 활용 가능      |
| `champions_vp`      | ⚠️ 선택   | 정적 상수로 관리 가능, 패치 시 수동 업데이트 |

### 3.4 외부 데이터 소스 (상세)

#### 티어/사용률/승률 — 공개 JSON 있음

| 소스                     | 접근 방식         | 갱신 주기   | URL                                                                              |
| ------------------------ | ----------------- | ----------- | -------------------------------------------------------------------------------- |
| eurekaffeine scraper     | GitHub Pages JSON | 매주 월요일 | `https://eurekaffeine.github.io/pokemon-champions-scraper/battle_meta.json`      |
| eurekaffeine 개별 포켓몬 | GitHub Pages JSON | 매주 월요일 | `https://eurekaffeine.github.io/pokemon-champions-scraper/pokemon/{dex_id}.json` |
| Showdown Tier            | 웹 스크래핑 필요  | 일 단위     | showdowntier.com                                                                 |

**eurekaffeine JSON 데이터 포함 항목:**

- 사용률, 무브, 아이템, 특성, 파트너 포켓몬

#### 추천 팀 구성 — 오픈소스 활용 가능

| 소스                   | 접근 방식              | 비고                       |
| ---------------------- | ---------------------- | -------------------------- |
| Andrew21P/ChampionsLab | GitHub TypeScript 파일 | 오픈소스, 팀 데이터 구조화 |
| Victory Road           | HTML 스크래핑          | 실전 검증 팀, 출처 명시    |
| PikaChampions          | 미확인                 | 팀 코드 공유 기능          |

#### VP 계산 공식 — 정적 관리 가능

커뮤니티 역산 공식 (Game8, GAMES.GG, Serebii 출처):

- 승리: +300 VP
- 패배: -150 VP
- 랭크 게이지: 승 1/4 충전, 패 1/4 감소
- 3연승 보너스: 충전 2/4로 상승

**관리 방식 제안:** 프론트엔드 상수로 관리, 패치 노트 모니터링 후 수동 업데이트

#### 기타 외부 리소스

| 소스          | 유형                    | 공개 API                |
| ------------- | ----------------------- | ----------------------- |
| Porygon Labs  | 데미지 계산기 + 팀 빌더 | ❌                      |
| ChampionsMeta | 메타 팀 + 팀 빌더       | ❌                      |
| PokéAPI       | 기본 도감 데이터        | ✅ (챔피언스 메타 없음) |
