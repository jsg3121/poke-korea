# BA-001: 포켓몬 챔피언스 기능 추가 경쟁력 분석

- 분석일: 2026-04-18
- 에이전트: business-analyst
- 참조: MI-001

## 핵심 인사이트 (Top 3)

1. **단기 선점 — VP 반영 데미지 계산기**: 기존 데미지 계산기 UI·로직을 유지하면서 VP 배율 파라미터를 추가하는 방식으로 구현 가능. 경쟁사에 한국어 대응 도구가 없어 검색 유입 직결 효과가 가장 높다.
2. **중기 차별화 — 포챔스 전용 도감 필터**: 186종 한정 수록, 메가/Z/다이맥스/테라스탈 폼 통합 표시, VP 등급별 스탯 비교. 기존 도감 인프라 재활용으로 개발 비용 최소화 가능.
3. **장기 투자 — 메타/사용률 통계**: 포챔스닷컴·OP.GG가 선점한 영역. API 수집 인프라 구축이 필요하므로 단기 우선순위에서 제외. 월드챔피언십 시즌(2026 하반기)을 타깃으로 준비.

## 전략 판단을 위한 미결 질문

1. VP 수치 데이터를 어떤 소스에서 확보할 것인가? (공식 API 없을 경우 커뮤니티 크라우드소싱 필요)
2. 포챔스 전용 페이지를 신규 라우트로 분리할 것인가, 기존 도구에 탭을 추가할 것인가? (SEO 구조 결정에 영향)
3. AdSense 수익 최적화 목표가 있는가? 트래픽 급증 시즌(월드챔피언십)에 맞춘 광고 배치 전략 검토 필요.

## SP 인풋 포인트

poke-korea의 실행 가능한 포지션은 **"포켓몬 챔피언스 한국어 계산기 허브"**다. 통계/메타 분석은 OP.GG가 장악하므로 직접 경쟁을 피하고, 도구(계산기·빌더) 영역에서 한국어 유일 서비스로 자리잡는 것이 현실적이다. 모바일 출시 전(2026 여름) 이전에 VP 계산기와 포챔스 도감 필터를 배포하는 것이 선점 타이밍의 핵심이다.
