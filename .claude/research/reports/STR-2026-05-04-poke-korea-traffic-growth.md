## Executive Summary

포케코리아의 최대 단기 기회는 **2026년 여름 포켓몬 챔피언스 모바일 출시**(한국닌텐도 공식 확정)이며, **챔피언스 개별 포켓몬 상세 페이지는 이미 구현되어 있어** 신규 구축이 아닌 **메타 description 다양화와 사이트맵 priority 상향**으로 모바일 출시 전 인덱싱 품질을 끌어올리는 것이 핵심이다. 네이버 실측 데이터에서 가장 즉시 개선 가능한 레버는 **노출 7,959회 vs 클릭 192회의 /detail/902(대쓰여너)** 같은 "노출은 많은데 CTR 2~3%인 페이지" 메타 description 최적화이며, /type-effectiveness 또한 동일 패턴(노출 7,431/CTR 2.0%)이다. 반면 /quiz/type-effectiveness는 동일 사이트 내에서 CTR 22.2%를 기록하며 페이지별 격차가 매우 크다는 점에서 **메타 품질이 직접 원인**임을 시사한다. **챔피언스 상세 페이지의 메타 description이 모든 포켓몬에 boilerplate 패턴**(`${pokemon.name}의 포켓몬 챔피언스 사용률, 인기 기술, 아이템, 특성, 추천 파트너 정보를 확인하세요.`)이라 검색엔진에서 thin content로 인식될 가능성이 30일 UV 3,494명 정체의 진짜 원인으로 추정된다. 권고 방향은 **"즉시 메타 최적화(전 페이지) + 챔피언스 메타·인덱싱 품질 개선 + 특수 폼/메가진화 콘텐츠 확장"의 3축 동시 실행**이다.

### Quick Wins (1~2주, 즉시 실행)

#### QW-1. 메타 description 전 페이지 네이버/구글 이중 최적화

**어떻게 (네이버 실측 우선순위 반영)**:

1. **최우선 (네이버 노출 5,000+ 페이지)**:

2. **차순위 (메가진화 페이지)**:
   - `/detail/[id]/mega` 라우트 메타데이터 — 메가진화 description 강화. 768/358/952 등
3. **3순위 (홈 + 챔피언스)**:
   - `src/app/_metadata/homeMetadata.ts` — 80자 이내 압축, 키워드 앞배치

4. **폴백 정비**: `src/app/layout.tsx` 기본 description 의미 있는 폴백으로 교체

**예상 효과**: 네이버 핵심 페이지 CTR 2% → 5%로 개선 시 주간 클릭 +800~1,200회 가능(추정). MAU 환산 약 3,500~5,000명/월 증가 가능성. 구글에도 동일 효과.

#### MT-5 (신설). 이로치(`?shinyMode=shiny`) URL 정적 경로 분리 검토

**무엇을**: `/detail/[id]?shinyMode=shiny` 쿼리 파라미터 URL을 `/detail/[id]/shiny` 정적 경로로 분리.

**왜**: 네이버 SC에서 이로치 페이지가 노출 상위 다수. 그러나 쿼리 파라미터 URL은 일반적으로 SEO에서 정적 경로 대비 약함(canonical 처리 따라 다름). 이로치 키워드 4개 TOP 10 진입 강세 활용.

**어떻게**:

1. 현재 `?shinyMode=shiny` 파라미터 URL의 canonical 설정 확인
2. `/detail/[id]/shiny/page.tsx` 정적 경로 신설 검토
3. 기존 쿼리 URL은 301 리다이렉트 또는 canonical로 정적 경로 지정
4. 메타 description: "{포켓몬명} 이로치(색이 다른 포켓몬) - 한국어 도감"

**주의**: 잘못된 마이그레이션은 기존 SEO 자산 훼손 위험 → 코드 변경 전 SEO 영향 분석 필수.

**예상 효과**: 이로치 페이지 CTR 2.4% → 5~10% 개선 가능성(추정). 정확한 효과는 마이그레이션 전후 비교 필요.

**우선순위 점수**: 영향도(중간) × 1/난이도(중간-높음) = **5/10**

**의존성**: SEO 영향 분석 필요. 라우팅 구조 변경 검토.
