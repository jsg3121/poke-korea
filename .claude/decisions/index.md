# decisions/

아키텍처 의사결정 기록(ADR)을 관리하는 폴더입니다.

## 사용법

1. 새로운 기술적 의사결정이 필요할 때 `template.md`를 참고하여 ADR을 작성합니다.
2. 파일명: `records/ADR-{번호}-{제목}.md` (예: `ADR-0001-harness-engineering.md`)
3. 기존 지침과 충돌하는 결정은 ADR 기록 후 관련 문서를 동기화합니다.
