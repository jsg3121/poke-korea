# skills/

커스텀 스킬을 정의하는 폴더입니다. 각 스킬은 `폴더/SKILL.md` 구조입니다.

## 개발 스킬

| 스킬                 | 유형        | 설명                              |
| -------------------- | ----------- | --------------------------------- |
| `/create-pr`         | 수동        | PR 생성 (조건부 검증 포함)        |
| `/lint-check`        | 수동        | Prettier + ESLint 전체 검사       |
| `/seo-audit`         | 수동        | SEO 메타태그/JSON-LD/sitemap 감사 |
| `/a11y-check`        | 수동        | WCAG 접근성 검사                  |
| `/code-review`       | 수동        | 서브에이전트 기반 코드 리뷰       |
| `/component-builder` | 수동        | Astro/Svelte 컴포넌트 생성        |
| `/research`          | 자동 트리거 | 외부 정보 조사 (공식 문서 우선)   |

## 테스트 스킬

| 스킬           | 유형 | 설명                       |
| -------------- | ---- | -------------------------- |
| `/test-writer` | 수동 | Vitest 단위 테스트 작성    |
| `/e2e-test`    | 수동 | Playwright E2E 테스트 작성 |

## 비즈니스 분석 스킬

| 스킬            | 유형        | 설명                                          |
| --------------- | ----------- | --------------------------------------------- |
| `/biz-strategy` | 자동 트리거 | 비즈니스 전략 분석 파이프라인 (MI → BA → STR) |

## 스킬 폴더 구조

```text
skills/
├── {skill-name}/
│   ├── SKILL.md              # 스킬 정의 (실행 절차, 출력 형식)
│   └── references/           # 참조 문서 (선택)
│       └── *.md
└── index.md                  # 이 파일
```
