# skills/

커스텀 스킬(슬래시 커맨드)을 정의하는 폴더입니다.

## 스킬 목록

| 스킬 | 유형 | 설명 |
|------|------|------|
| `/create-branch` | 수동 | 신규 버전 브랜치 생성 |
| `/create-pr` | 수동 | Pull Request 생성 |
| `/review-pr` | 수동 | PR 코드 리뷰 코멘트 분석 |
| `/reply-review` | 수동 | PR 리뷰 코멘트 답글 작성 |
| `/lint-check` | 수동 | Prettier + ESLint 전체 검사 |
| `/seo-audit` | 수동 | SEO 메타태그/JSON-LD/sitemap 감사 |
| `/code-review` | 수동 | 서브에이전트 기반 코드 리뷰 |
| `/component-builder` | 수동 | desktop/mobile 컴포넌트 스캐폴딩 |
| `/feature-plan` | 수동 | 신규 기능 기획서 작성 (요구사항/설계 정리) |
| `/page-scaffold` | 수동 | 새 페이지 풀 세트 스캐폴딩 (page + views + container + context + SEO) |
| `/competitive-audit` | 수동 | 경쟁 사이트 특정 기능 비교 분석 |
| `/research` | 자동 트리거 | 외부 정보 조사 (공식 문서 우선) |
