# 핵심 지표 정의

## North Star Metric

| 지표 | 정의 | 측정 방법 |
|------|------|-----------|
| MAU | 월간 활성 사용자 수 | Google Analytics 기준 30일 내 1회 이상 방문한 고유 사용자 |

## AARRR 퍼널 지표

### Acquisition (획득)

| 지표 | 정의 | 산식 |
|------|------|------|
| 일일 방문자 수 (DAU) | 하루 고유 방문자 | GA 세션 기준 |
| 채널별 유입 비율 | 유입 경로별 비중 | 채널별 세션 / 전체 세션 |
| 검색 노출 수 | 검색 결과에 노출된 횟수 | Search Console Impressions |
| 검색 클릭률 (CTR) | 노출 대비 클릭 비율 | Clicks / Impressions |

### Activation (활성화)

| 지표 | 정의 | 산식 |
|------|------|------|
| 페이지뷰 / 세션 | 세션당 평균 페이지 조회 수 | 총 페이지뷰 / 총 세션 |
| 도구 사용률 | 도구/계산기를 사용한 세션 비율 | 도구 사용 세션 / 전체 세션 |
| 이탈률 (Bounce Rate) | 단일 페이지 방문 후 이탈 비율 | 단일 페이지 세션 / 전체 세션 |

### Retention (리텐션)

| 지표 | 정의 | 산식 |
|------|------|------|
| D1 리텐션 | 다음날 재방문율 | D1 재방문 사용자 / D0 신규 사용자 |
| D7 리텐션 | 7일 후 재방문율 | D7 재방문 사용자 / D0 신규 사용자 |
| D30 리텐션 | 30일 후 재방문율 | D30 재방문 사용자 / D0 신규 사용자 |
| 재방문율 | 기존 방문자 비율 | 재방문 사용자 / 전체 사용자 |

### Revenue (수익)

| 지표 | 정의 | 산식 |
|------|------|------|
| 광고 수익 (AdSense) | Google AdSense 수익 | AdSense 대시보드 |
| RPM | 1,000회 노출당 수익 | 수익 / (페이지뷰 / 1000) |

### Referral (추천)

| 지표 | 정의 | 산식 |
|------|------|------|
| 공유율 | 콘텐츠 공유 비율 | 공유 이벤트 / 세션 |
| 백링크 수 | 외부 사이트에서 링크된 수 | Search Console 링크 보고서 |
