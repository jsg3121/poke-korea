# 신뢰할 수 있는 데이터 소스

market-intelligence 에이전트가 시장 조사 시 참조하는 데이터 소스 목록이다.

| 소스 | URL | 설명 |
|------|-----|------|

| State of CSS | stateofcss.com | CSS 생태계 연간 설문 |
| npm trends | npmtrends.com | npm 패키지 다운로드 비교 |
| JetBrains Developer Survey | jetbrains.com/lp/devecosystem | 개발 도구/언어 사용 현황 |
| Statista - Developers | statista.com | 전 세계 개발자 수, 기술 시장 규모 |
| SlashData - Developer Nation | slashdata.co | 글로벌 개발자 인구 통계 |

## 데이터 사용 원칙

1. **최신 데이터 우선**: 1년 이내 자료를 우선 사용. 2년 이상 된 데이터는 트렌드 변화를 감안하여 주석 추가
2. **교차 검증**: 단일 출처에 의존하지 않고 최소 2개 이상 소스로 교차 확인
3. **유료 데이터 한계 명시**: 접근 불가한 유료 데이터는 "유료 소스 참조 필요"로 명시
4. **추정값 구분**: 직접 데이터와 추정값을 명확히 구분 표기
